# CRO AI Toolkit

## Projektstruktur

- `app.py` – Startpunkt der Flask-App
- `templates/` – HTML-Templates (z.B. index.html)
- `static/` – Statische Dateien (CSS, JS, Bilder)

## Setup & Start

1. Installiere die Abhängigkeiten:
   ```bash
   pip install flask
   ```
2. Starte die App:
   ```bash
   python app.py
   ```
3. Öffne [http://127.0.0.1:5000](http://127.0.0.1:5000) im Browser.
